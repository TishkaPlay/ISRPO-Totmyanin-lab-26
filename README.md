# 📝 Менеджер задач (Task Manager)

## Описание
Простое веб-приложение для управления задачами с возможностью добавления, отметки выполнения, фильтрации и сохранения данных в браузере.

## Функционал
- ✅ Добавление задач (кнопка или клавиша Enter)
- ✅ Отметка выполненных задач (чекбокс)
- ✅ Удаление задач с мгновенным обновлением списка
- ✅ Фильтрация: Все / Активные / Выполненные
- 💾 Автоматическое сохранение в `localStorage` (данные не теряются при перезагрузке)
- 📊 Статистика: общее количество и количество выполненных задач

## Технологический стек
- HTML5 / CSS3 (Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Классы, Arrow functions, Template literals
- Делегирование событий, `localStorage` API
- Защита от XSS (`escapeHtml`)

## Как запустить
1. Откройте папку проекта в VS Code.
2. Установите расширение **Live Server**.
3. Нажмите `Go Live` в правом нижнем углу или кликните правой кнопкой по `index.html` → `Open with Live Server`.
4. Приложение откроется в браузере и готово к использованию.
